# rustfluent

A Python interface to the Rust Fluent Library.

This project is a small shim around [fluent-rs](https://github.com/projectfluent/fluent-rs), so it
can be used from Python.

## Prerequisites

This package supports:

- Python 3.10
- Python 3.11
- Python 3.12
